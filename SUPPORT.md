# Support

Agentic is maintained as an open-source project. Support happens through public project channels unless the topic is security-sensitive.

## Questions And Usage Help

Use GitHub issues for reproducible problems, documentation gaps, and focused support questions:

https://github.com/leonardwongly/agentic/issues

Before opening an issue, include:

- what you were trying to do
- the exact command, route, or workflow involved
- relevant environment details such as Node version and persistence mode
- sanitized logs or error messages
- whether the worker was running for queued flows

## Bugs

Use the bug report issue template and include a minimal reproduction whenever possible.

## Security

Do not use public issues for vulnerabilities or sensitive operational data. Follow [SECURITY.md](SECURITY.md).

## Commercial Or Private Support

No commercial support channel is currently published. Project maintainers may decline private support requests that cannot be handled safely or transparently.
